# Contributing

## Development Setup

```bash
npm install
cp .env.example .env
npm run dev
```

This runs the client (Vite) and server (Express) concurrently.

## Testing Locally

```bash
npm run test
```

## Before Submitting PR

- [ ] `npm run lint` passes
- [ ] `npm run format:check` passes
- [ ] `npm run test` passes
- [ ] No console errors
